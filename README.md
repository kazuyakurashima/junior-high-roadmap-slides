# 中学入試ロードマップ スライド

茨城県の小学4・5年生の保護者向けの「中学入試ロードマップ」セミナー用のインタラクティブスライドデッキです。

## 概要

- 40分間のオンラインセミナー用スライド
- HTML、TailwindCSS、shadcn/uiを使用
- データ可視化を含む18枚のスライド
- レスポンシブデザイン（モバイル・デスクトップ対応）
- 日本語コンテンツ

## 開発方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## デプロイ

Vercelを使用して静的サイトとしてデプロイできます。
